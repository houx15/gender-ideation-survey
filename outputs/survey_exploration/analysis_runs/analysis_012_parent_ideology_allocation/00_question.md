# analysis_012 — Does parent gender ideology drive gendered allocation? (CFPS, SPEC 5.5)

**Research question:** Do more traditional parents allocate resources in a son-favouring
way *within the family* — giving daughters relatively less education (investment) and
demanding relatively more housework from them (demand)?

**Why CFPS (and a data boundary):** the rich resource data in CEPS (analysis_011) cannot
be linked to gender ideology — CEPS has **no parental gender-attitude item** and only
**anonymized county codes** (no province), so neither a direct measure nor a regional
ideology-climate merge is possible. CFPS is the only dataset with **both** measured
parental ideology and parent–child links, so the direct test runs here.

**Analysis type:** Intergenerational reproduction; within-family (one-son-one-daughter) design.

**Data / sample:** CFPS 2014 & 2020. One-son-one-daughter families (367/385 with education;
385/177 with both children's housework; 107/189 with both siblings ≥25).

**Core variables:** within-family daughter−son gap in years of schooling (investment) and
in housework hours (demand); family `parent_mean` ideation ([0,1], 1 = most traditional);
`age_gap` control.

**Predicted son-favouring signs:** education gap coefficient **negative**, housework gap
coefficient **positive**.
