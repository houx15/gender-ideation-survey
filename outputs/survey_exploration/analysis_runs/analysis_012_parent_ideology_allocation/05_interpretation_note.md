# analysis_012 — Interpretation note

## What the results say (RQ 5.5)
From `04_result_table.csv` / `01_descriptive_table.csv` (one-son-one-daughter families).

**Directionally son-favouring, but not statistically significant.**
- **Education gap (daughter − son):** the parent-ideology coefficient is **negative** in
  most specifications (2014 all-ages **−2.71, t=−1.63**; 2020 −0.17 / −0.47) — more
  traditional parents → daughter gets relatively *less* schooling — but only the 2014
  all-ages case is even marginal, and it weakens at age ≥ 25.
- **Housework gap (daughter − son):** the coefficient is **positive** in both waves
  (**+0.94, t=1.09** in 2014; **+1.11, t=0.85** in 2020) — more traditional parents →
  daughter does relatively *more* housework — but neither is significant.

So **both channels point the predicted way (son-favouring under traditional parents), yet
none clears conventional significance.** The within-family one-son-one-daughter samples
(107–385 families) are simply underpowered to detect gaps of this size.

## Honest bottom line on "how parental ideology influences allocation"
- With the available data we **cannot confirm** that parental gender ideology produces
  measurably gendered resource allocation. The signs are consistent with the hypothesis;
  the evidence is not statistically conclusive.
- The strong, *significant* allocation patterns we did find (CEPS analysis_011 — girls get
  more educational investment overall but lose the tutoring-spending advantage once they
  have a brother) **cannot be attributed to parental ideology**, because CEPS carries no
  ideology measure and no geography to merge a climate proxy.

## The data gap this exposes (most useful takeaway)
A clean answer needs **parental gender ideology linked to during-childhood resource
allocation**. Neither file we have provides both: CEPS has the resources but no ideology;
CFPS has ideology but its child resources are limited to selected, mostly-adult co-resident
children. The unlock is a **CFPS children's questionnaire** (per-child expenditure,
expectations, tutoring) merged to the parent's ideology — see the readiness report.

## Next steps

mixed-gender sibships; if any survey with both measures exists, replicate there.
